HTML

The <html> element tells the browser that it is an HTML document. It follows the <!DOCTYPE html> tag.

Syntax:

<html> </html>

Header

The <head> element indicates a section of code that contains metadata about the HTML document. It is usually placed between the <html> and <body> tags.

Syntax:

<head> </head>

Title

The <title> element declares the title of the HTML document. This is included as part ofthe metadata in the header element.

Syntax:

<title> </title>

Body

The <body> element defines the main, visible content of the HTML document. The majority of your coding is included here.

Syntax:

<body> </body>

Headings

The <h1> element tells the browser that it is an HTML heading. You can use <h1> to <h6> to indicate a heading's level of importance.

Syntax:

<h1> </h1>

Example:

This is a heading.


Paragraph

The <p> element indicates an HTML paragraph. You can include numbers after the "p" in the tag in distinguish different paragraphs from each other.

Syntax:

<p> </p>

Example:

This is a paragraph.


Breaks

The <br> element declares a line break within the text.

Syntax:

<br> </br>

Example:

This is a sentence
with a break.

Lines

The <hr> element defines a break. It is used to separate content by drawing a horizontal line across an HTML document.

Syntax:

<hr> </hr>

Example:

Hi
Bye

Comments

The <!-- --> element tells the browser that it is a comment. The text within this element will not be visible in the actual browser.

Syntax:

<!-- -->

Links

The <a> element indicates a link in the HTML document. It is often used with the <a href> attribute which specifies a destination address.

Syntax:

<a> </a>

Example:

Go to w3schools

Images

The <img> element declares an image in the HTML document. It is an empty tag, so it does not require a closing tag.

Syntax:

<img>

Example:


Lists

The <ul>, <ol>, and/or <li> elements defines an unordered list, ordered list, and/or item in a list, respectively. Within these tags, you can specify the list markers.

Syntax:

<ul> </ul>

Example:

  • Item 1
  • Item 2
  • Item 3

Tables

The <table> element tells the browser that it is a table. Within this tag, there are <tr> tags for each row, <th> tags for each table header, and <td> tags for each table "cell."

Syntax:

<table> </table>

Example:

Column 1 Column 2 Column 3
a b c
d e f

Blocks

The <div> element is a block-level element that separates a certain section of code. It is usually used along with style and class attributes.

Syntax:

<div> </div>

Example:

Hello
World
